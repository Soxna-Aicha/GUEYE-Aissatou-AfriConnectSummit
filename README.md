## Description du projet

AfriConnect Summit est un site web vitrine réalisé dans le cadre de l'examen de Technologie Web.

Le projet présente une conférence technologique fictive réunissant des développeurs, des entrepreneurs et des investisseurs africains autour de l'innovation numérique.

Le site est entièrement responsive et comprend plusieurs fonctionnalités interactives développées en HTML5, CSS3 et JavaScript Vanilla.

## Fonctionnalités JavaScript

- Mode sombre / mode clair avec sauvegarde dans le localStorage
- Navbar dynamique au défilement
- Menu hamburger responsive
- Compte à rebours jusqu'à la conférence
- Compteurs animés
- Animations au scroll avec IntersectionObserver
- Onglets dynamiques du programme
- Filtrage des intervenants
- Validation du formulaire d'inscription
- Bouton Retour en haut
- Année dynamique dans le footer
